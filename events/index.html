This is a paragraph of content. Notice the p tags.

You can create another paragraph of content. Notice how they have space between each paragraph.

You can format text. You can make things bold by using the strong tags. You can also italicize text using the emphasis (em) tags.

Do you need a hyperlink? Here's a hyperlink to ConnectVA. The href attribute defines where to go. If you want the hyperlink to open in a new, blank tab; simply use target="_blank".

How about an image? Use the img tag. Just reference the name of the image in the media folder like you see below. Pay attention to the number of slashes.

Finally, how about a list? Here's how you can create a list:

Unordered List:
  • First
  • Second
  • Third
Ordered List:
  1. First
  2. Second
  3. Third

UL for unordered lists. OL for ordered lists. LI for list items.
